# AI Recommendation System

## Project Overview

The AI Recommendation System is a Python-based application that provides personalized recommendations based on user preferences. The system uses similarity matching and pattern recognition techniques to analyze user interests and recommend the most relevant items.

This project was developed as part of the DecodeLabs Artificial Intelligence Internship Program. It demonstrates the fundamental concepts of recommendation systems, which are widely used in real-world platforms such as Netflix, Amazon, Spotify, and YouTube.

## Objective

The main objective of this project is to:

- Collect user preferences as input
- Analyze and compare preferences with available data
- Calculate similarity scores
- Generate personalized recommendations
- Display ranked recommendations based on relevance

## Features

- User-friendly command-line interface
- Multiple preference selection
- Similarity score calculation
- Pattern matching algorithm
- Ranked recommendation system
- Personalized recommendations
- Clean and modular Python code

## Technologies Used

- Python
- Data Structures
- Set Operations
- Similarity Matching
- Pattern Recognition
- Recommendation Logic

## How It Works

1. The user enters their interests or preferences.
2. The system compares the user preferences with predefined item categories.
3. A similarity score is calculated based on matching attributes.
4. Items are ranked according to their scores.
5. The top recommendations are displayed to the user.

## Example

### Input

```
Sci-Fi, Space
```

### Output
